Вижу 4 пути решения задачи если использовать только это скобки:

1. Заменить ВСЕ скобки на круглые с сохранением их направленности;
2. Заменить ВСЕ скобки на квадратные с сохранением их направленности;
3. Заменить ВТОРУЮ скобку СЛЕВА на квадратную с сохранением направленности;
4. Заменить вторую скобку СПРАВА на круглую с сохранением направленности.
